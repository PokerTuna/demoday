# SETTING UP
In root folder run:
```
cp wp-config-sample.php wp-config.php
docker-compose up -d
yarn install
yarn run dev
```

then open `http://localhost:8000/`, setup DB, login to wp-admin, select `Packhelp Demo Day` theme and it's done.
PHPMyAdmin is available under `http://localhost:8080/`
